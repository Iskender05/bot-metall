# mir-metalla-bot

Чат-бот для компании «Мир металла». Один входящий текст клиента сохраняется в БД, затем отправляется в LLM вместе с историей диалога и единым системным промптом.

## Основное

- Системный промпт: `src/agent/mir_metalla_system_prompt.py`
- История диалога: таблица `messages`
- Статус диалога: поля `users.answers_from_agent` и `users.final_stage`
- Если менеджер вручную пишет клиенту в Avito, бот сохраняет это сообщение и больше не отвечает в этом диалоге
- Для теста можно ограничить ответы одним или несколькими Avito chat_id через `AVITO_ALLOWED_CHAT_IDS`

## Запуск

1. Скопируйте `example.env` в `.env` и заполните токены.
2. Запустите сервисы:

```bash
docker compose up --build
```

Telegram по умолчанию отключён. Чтобы включить его отдельно, задайте `ENABLE_TG_BOT=True` и `TG_BOT_TOKEN`.
